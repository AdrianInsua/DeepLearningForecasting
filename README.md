### Requirements

- KERAS

### Installation

set privileges to install.sh

```
chmod +x install.sh
```

execute installation script

```
./install.sh
```

### Configuration

You can change configuration variables in config.py file

### Basic Usage

to start just execute the basic script with default configuration

```
python main.py
```
